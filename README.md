## Що таке n8n
**n8n** — це платформа для автоматизації (workflow automation), де ти збираєш сценарії з блоків (nodes):  
отримав дані → обробив → передав далі (таблиці, месенджери, CRM, AI тощо).

## Важливо перед стартом
Під час роботи з таблицями та інтеграціями інколи трапляються збої, тому:
- після входу в n8n **завжди перевіряй версію в адмінці**;
- **використовуй останню стабільну версію**, а не просто “найновішу”;
- бажано **фіксувати версію** (щоб оновлення не ламали воркфлоу).

## Що потрібно для роботи

### 1) AI-модель для тексту (ChatGPT / OpenAI)
1. Перейди на https://platform.openai.com/  
2. Створи акаунт  
3. Поповни баланс **мінімум на $5** (щоб запити почали оброблятися)  
4. Створи **API key** і підключи його в n8n (Credentials)

### 2) Сервіс для генерації та зберігання медіа (фото/відео) — fal.ai
n8n **не є сховищем** для фото/відео, тому потрібен зовнішній сервіс, який:
- генерує контент (або приймає/зберігає його),
- повертає **URL/файл**, з яким n8n може працювати далі.

**Приклад:** https://fal.ai/  
**Ціна:** оплата **за запити (pay-per-request)** — кожна генерація фото/відео/музики коштує грошей.

**Важливо:**
- поповни баланс **мінімум на $5** (щоб запити почали оброблятися)
- при збоях/ретраях може списати кошти кілька разів;
- бажано ставити ліміти та обмежувати повтори в n8n.
Особистий орієнтир: інколи потрібно ~**$40**, щоб “пристрілятися” і зібрати стабільний пайплайн.

### 3) Вартість n8n
n8n має **два формати використання**:

- **Безкоштовно** — якщо розгорнути n8n **на власному сервері** (self-hosted). Це open-source версія без абонплати, ти платиш лише за свій хостинг.
- **Платно** — якщо користуватися **n8n Cloud** і не заморочуватися з сервером, оновленнями та підтримкою:
  - приблизно **€24/міс + податок** (≈ **€28.80/міс**);
  - **пробний період — 14 днів**.
